## Reverse Words in a String

Hint: String methods

- Use the String split()
- Reverse the string that way

``` java
class Solution {
    public String reverseWords(String s) {
        String[] arr = s.split(" ");
        String ans = "";

        for (int i = arr.length - 1; i >= 0; i--) {
            ans += arr[i];

            if (i != 0 && arr[i] != "") {
                ans += " ";
            }
        }

        return ans.strip();
    }
}
